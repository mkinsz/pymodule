## Create and Activate the virtual environment

> In Windows:
> py -3 -m venv .venv
> .venv\scripts\activate
> ...
> In MacOS/Linux:
> python3 -m venv .venv
> source .venv/bin/activate

## Install the packages

> py -3 -m pip install ***
